ChatMeii

ChatMeii 是一个网页角色聊天模拟器。
你可以选择当前扮演者，以与你的小梅聊天、互动、角色扮演，甚至演锦衣卫。
———— 现在、过去、未来。命运要如何转动，全取决于你。

基本玩法

  • 在底部输入内容，选择当前扮演者后发送。
  • 你可以作为“自己”对话，也可以切换为朱瞻基、马醉灯、风舞阳、侯传玉等角色推进剧情。
  • 左侧会显示当前状态，包括心情、创伤、依恋、兴奋、执势、风声和角色关系。
  • 开启“场景描写”后，系统会在适当时候补充一小段场景余韵。
  • 如果不知道从哪里开始，可以直接和梅霜花说一句话，或用角色身份发起剧情。

左侧设置

  • 自定义设定：修改当前时间线、背景设定和人物补充。
  • 数据管理：清空对话、回滚上一轮、存档、读档、导出 TXT、导出完整备份或恢复完整备份。
  • 模型配置：填写 API Key、Base URL 和模型名。
  • 显示设置：调整深色/浅色模式、对话字体大小、场景描写显示等界面选项。
  • 调试工具：查看或手动调整状态数值，适合测试不同剧情分支。

数据提醒

  • 对话、存档和设定主要保存在当前浏览器中。
  • 清理缓存、更换浏览器或更换设备后，数据可能丢失。
  • 建议定期导出完整备份。
  • API Key 仅保存在本地浏览器，请自行保管，不要分享给他人。
